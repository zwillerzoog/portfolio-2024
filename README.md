# Welcome!
You've stumbled upon the Portfolio website for Heidi Marsh. This is a [Next.js](https://nextjs.org/) project bootstrapped with [`create-next-app`](https://github.com/vercel/next.js/tree/canary/packages/create-next-app).

This project uses [`next/font`](https://nextjs.org/docs/basic-features/font-optimization) to automatically optimize and load Inter, a custom Google Font.

## Prerequisites

Node v18.17.0 or higher

## Getting Started

From the terminal:
```sh
 git clone https://github.com/zwillerzoog/remix-hello-world.git
```

```sh
cd remix-hello-world
```

```sh
npm install
```

Then run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
